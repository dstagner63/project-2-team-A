# Demo template

## Setup

Run the generate_db script:
```
conda activate bcs
cd data
python generate_db.py
```

You can do more data cleaning/joining/transformations in this step, as needed (same idea as ETL project).

## Run

```
conda activate bcs
python app.py
```
